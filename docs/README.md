# Documentation

Эта папка содержит документацию проекта.

## Файлы

- `SYSTEM_REQUIREMENTS.md` - системные требования для запуска приложения
- `CI_CD_VARIABLES.md` - руководство по управлению переменными окружения в CI/CD
- `TROUBLESHOOTING.md` - решение типичных проблем

## Содержание

### SYSTEM_REQUIREMENTS.md

Содержит информацию о:
- Необходимых системных зависимостях
- Требованиях к Python
- Установке LaTeX и LibreOffice
- Настройке окружения

### CI_CD_VARIABLES.md

Содержит информацию о:
- Как правильно добавлять новые переменные окружения
- Структуре CI/CD конфигурации
- Чеклисте при добавлении переменных
- Примерах использования
